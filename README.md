# Manual
Консольное приложение - клиент, которое взаимодействует с сервером. 
Пользователь управляет клиентом с помощью команд
Список команд живет в модуле Client, в файле help.md

# Стак
- Netty-based server
- Netty-based client
- SQLlite db
